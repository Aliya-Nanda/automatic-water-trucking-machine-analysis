# 🚚 Automatic Water Trucking Machine Performance Analysis

MATLAB-based engineering performance analysis for the **Automatic Water Trucking Machine**, an autonomous prototype developed to simulate clean water distribution in disaster response scenarios.

This project was developed as part of the **Introduction to Engineering Design** course at the **Faculty of Mechanical and Aerospace Engineering, Institut Teknologi Bandung (ITB)**.

---

## 📌 Project Overview

The MATLAB program evaluates the engineering performance of the prototype by calculating multiple design metrics and assessing whether the system satisfies the project requirements.

The analysis supports engineering decision-making through quantitative evaluation and automated result visualization.

---

## 🎯 Objectives

- Evaluate prototype performance using engineering metrics
- Verify compliance with design requirements
- Analyze payload efficiency and mission performance
- Automate engineering calculations
- Generate visual summaries for design evaluation

---

## ⚙️ Features

- Engineering performance calculations
- Automated KPI evaluation
- Design requirement verification
- Performance visualization
- HTML report generation

---

## 📊 Performance Metrics

The program evaluates several engineering indicators, including:

- Payload Ratio
- Structural Weight Ratio
- Mission Completion Time
- Water Delivery Capacity
- Design Requirement Compliance
- Overall Performance Score
